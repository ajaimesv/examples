## .vimrc

Make sure you create this file before even starting vim or you will face [this error/feature](https://github.com/vim/vim/issues/1326).

```
set sm
set number
set ignorecase
set tabstop=4
set shiftwidth=4
set expandtab
set autoindent
set showmatch
syntax on
set ruler
set hlsearch
set smartindent
set cindent
```
